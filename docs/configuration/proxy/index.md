# Proxy

Proxy settings for Halloy.

> 💡 [Preview](../preview/) requests will be routed through the same proxy that the corresponding message is routed through (i.e. if a proxy is configured for a server, then all previews for messages on that server will be routed through the proxy).  Except for the for the [Tor](tor.md) proxy;  when utilizing the Tor proxy preview requests are disabled.
