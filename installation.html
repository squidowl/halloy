Build from source

Clone the Halloy GitHub repository into a directory of your choice and build with cargo.

Requirements:

  • Rust toolchain
  • Git version control system
# Clone the repository
git clone https://github.com/squidowl/halloy.git

cd halloy

# Build and run
cargo build --release
cargo run --release
