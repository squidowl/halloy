# Scale factor

```toml
scale_factor = <float>
```

| Key            | Description                                             | Default |
| -------------- | ------------------------------------------------------- | ------- |
| `scale_factor`[^1] | Control the scale factor of the entire application. | `1.0`[^2]   |


[^1]: `scale_factor` is a root key, so it must be placed before any section.
[^2]: Limited between `0.1` and `3.0`.
